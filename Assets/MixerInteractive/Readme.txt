Thank you for downloading the Mixer Interactive SDK. The SDK includes:
* Examples - Example scenes and instructions that demonstrate the functionality in this SDK.
* Prefabs - The InteractivityManager prefab which you can use to talk to the Mixer Service
* Source - The DLLs, helper scripts and editor extensions for the SDK

There are instructions included in each of the example scenes.

You can learn more about the Mixer SDK at: https://dev.mixer.com/

For bugs or other issues, you can find support at the official GIT repo: https://github.com/mixer/interactive-unity-plugin/issues

For feedback, you can visit our feedback site: https://feedback.mixer.com/forums/382521-lazers-and-shiny-things
